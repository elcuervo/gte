   - Dimension sanity check: embed a single test string with both, print dimensions, abort if they differ

     - `x.report("fastembed") { fe_model.embed(texts).to_a }`
     - `x.compare!`
   - Print batch size header before each Benchmark.ips block: `puts "\n=== Batch size: #{texts.size} ==="`

3. Make the script executable: `chmod +x bench/benchmark.rb`

4. Run `bundle install` to install the new dependencies.
